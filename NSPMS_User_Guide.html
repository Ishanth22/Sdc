🟢 Persona 1 — Arjun Mehta (Healthy Founder)

Email: founder1@nspms.in | Password: password123
Startup: NovaSaaS Technologies | SaaS · Series A · Bengaluru · 45 employees
Mock Data: 8 months of metrics (Aug 2024 → Mar 2025), ₹8Cr total funding

Step-by-Step Walkthrough

1. Login

  1. Go to http://localhost:5173
  2. Enter founder1@nspms.in / password123 → click Login
  3. ✅ Redirected to /dashboard

Test Account Lock: Enter wrong password 5 times → see the lockout error. Wait 15 min or run npm run seed to unlock.

2. Dashboard (/dashboard)

  • Vitality Score Gauge — should show ~82–88/100 (green)
  • Risk Badge — Low 🟢
  • Funding Readiness — >80%
  • KPI Cards: Revenue ₹6.48L, Burn ₹0 (profitable), Runway 26 months, Users 3,350, Churn 3.1%, LTV/CAC 18.7x
  • 5 Charts: Revenue Growth (upward trend), User Growth (bar chart), Revenue vs Expenses (converging), Burn Rate Trend, Expense Distribution (pie)
  • Benchmark bar: Compare NovaSaaS vs industry averages (should be well above)
  • Alerts panel: Should show 0 critical alerts (healthy startup)

3. Submit Metrics (/metrics/new)

  1. Click "+ Submit Metrics" in the nav
  2. Select period → e.g., 2025-04
  3. Fill Financial: Revenue 720000, Expenses 535000, Burn 0, Runway 28, Funding 8000000
  4. Fill Operational: Active Users 3800, New Users 780, CAC 890, LTV 17800, Churn 2.9, GMV 2160000, Cities 14
  5. Fill Innovation: Patents Filed 5, Granted 2, Trademarks 4, R&D 118000
  6. Fill Impact: Direct Jobs 48, Women 19, Rural 8, Exports 650000
  7. Click Submit
  8. ✅ Redirected to Dashboard — charts update with new month, new vitality score computed

4. Metrics History & Audit Trail (/history)

  • Table shows all 8+ submitted periods in reverse order
  • Each row: Period, Revenue, Burn, Runway, Users, Churn, CAC, LTV/CAC, Health Score
  • Audit Log tab — see log of all actions (metrics submitted, profile saved, alerts dismissed)

5. Startup Profile (/profile)

  1. Click profile icon or nav → Profile
  2. See pre-filled: NovaSaaS Technologies, SaaS, Series A, Bengaluru, 45 employees
  3. Edit Company Description → save → ✅ Profile updated toast
  4. Edit Twitter/LinkedIn links → save

6. Milestones & OKRs (/milestones)

  1. View All tab — 9 milestones shown
  2. Click Milestones Only — 5 simple milestones, 5 with ✅ Completed badge
  3. Click OKRs Only — 3 OKRs with Key Results and progress bars
  4. Add a new milestone:
    • Click + Add Milestone
    • Title: Launch API Marketplace, Category: Product, Deadline: 2025-06-30, Description: any text
    • Toggle OKR mode → add Key Result: API Partners, Target 20, Unit partners
    • Save → appears in the list
  5. Update progress: drag the completion slider on any in-progress milestone
  6. Mark complete: click ✓ Complete on the new milestone → timestamp shown
  7. Overall progress bar at top updates automatically

7. Custom KPI Builder (/custom-kpis)

  1. See 4 pre-built KPIs: LTV/CAC, Revenue per Employee, Burn Multiple, NRR
  2. Each shows a line chart trending across 8 months
  3. Add a new KPI:
    • Click + Add KPI
    • Name: GMV per City, Formula: gmv / citiesServed, Unit: ₹, Description: any
    • Save → card with empty chart appears
  4. Log a value for a period:
    • Click + Log Value on the new KPI
    • Period: 2025-03, Value: 154615
    • Save → chart updates

8. AI Forecasting (/forecasting)

  1. Navigate to Forecasting
  2. Wait for AI to compute (first visit ~5–10 seconds with OpenRouter key)
  3. See 🤖 AI-Enhanced badge + cache timestamp
  4. 4 charts: Revenue Forecast, Expense Forecast, User Growth, Runway Projection
    • Solid lines = historical actual data (Aug 2024 → Mar 2025)
    • Dashed lines = AI predictions (Apr → Sep 2025)
  5. AI Insights panel — 5 bullet points of AI analysis
  6. Strategic Outlook — 2–3 sentence narrative
  7. Critical Action — single most important 30-day action
  8. Confidence level badge (should be high with 8 months of data)
  9. Submit new metrics → revisit Forecasting → AI regenerates (cache invalidated)

9. AI Intelligence Center (/ai-risk)

Tab 1 — Risk Prediction:

  1. AI Risk Score displayed (~25–35 for NovaSaaS — Low risk)
  2. Trend Analysis paragraph
  3. Risk Predictions grid — 6 cards with probability, timeframe, impact
  4. AI Recommended Actions — numbered list
  5. Click 🔄 Re-analyze button → forces fresh AI call (bypasses cache)

Tab 2 — AI Benchmarking:

  1. Competitive Position — should show "Top Quartile" or "Above Average"
  2. Peer Comparison summary sentence
  3. Strength Areas — metrics where NovaSaaS leads peers (green badges with + %)
  4. Areas to Improve — red badges with - %
  5. Competitive Edge narrative
  6. AI-Detected Opportunities

10. AI Advisor (/advisor)

  1. Auto-recommendations panel loads on page open (5 pre-generated cards)
  2. Type a custom question: "When should we start our Series B fundraise?"
  3. Press Send → AI responds with data-grounded answer citing actual numbers
  4. Ask follow-up: "What is our biggest operational risk right now?"
  5. Ask: "How do we reduce CAC from ₹920 to ₹700?"
  6. Data Points Used chips appear below each AI answer

11. Scenario Simulation (/simulation)

  1. Sliders: Revenue Change +30%, Expense Reduction 10%, Additional Funding ₹50L, Churn Improvement 20%
  2. Click 🔬 Run Simulation
  3. Radar Chart — amber shape (simulated) vs gray (current) — revenue and runway axes should expand
  4. Comparison Table — Current vs Simulated vs Change for all 6 metrics
  5. Try extreme: Revenue −40% → see health score drop sharply → Risk Level changes to High/Critical

12. Reports & Export (/reports)

  1. Click Performance Summary Report card → report generates in-browser
  2. See: company header, health score, latest KPIs, all historical metrics table, milestones, alerts
  3. Click 🖨️ Print / Save as PDF → browser print dialog opens; save as PDF
  4. Click 📄 Export CSV → .csv file downloads with all metrics history (open in Excel)

13. Logout

  • Click user avatar → Logout → redirected to /login
